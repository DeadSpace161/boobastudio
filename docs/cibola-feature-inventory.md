The requested parity list is therefore broad but not uniformly available locally. Wall detection, UI, document integration, queue mechanics, prompt/model forms, macro/radial menu, and Foundry file handling are local. Text/image/audio, gallery, packs, threads, and vectors are primarily hosted behind `AuthService`; AI document translation is deferred by user scope, while module UI localization remains local.
